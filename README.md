### 🚀 Destaque de Projeto: Auto About
**No-Code Chatbot Builder:** Liderei o desenvolvimento de uma ferramenta de gestão visual para fluxos de WhatsApp utilizando **React Flow** e **Gemini API**. 
- **O Desafio:** Permitir que usuários não técnicos criem automações complexas.
- **A Solução:** Uma interface drag-and-drop que mapeia nós visuais em árvores de decisão processadas em **Node.js**.
